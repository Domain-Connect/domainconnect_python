# Security Policy

## Supported Versions

These versions of the project and python versions are currently being supported with security updates.

|Python Version| Version | Supported          |
| ------- | ------- | ------------------ |
| >= 3.7  | 0.0.11   | :white_check_mark: |
| >= 3.7  | <0.0.11   | :x:                |
| < 3.7  | any   | :x: |
